# NOTE FOR DEVELOPERS

The JavaScript code in this distribution has been compiled, bundled, and
minified from its original TypeScript source. As a result, it is no longer
human-readable or directly editable.

If you wish to:

- Read the source code
- Contribute to the project
- Report a bug or open an issue
- Fork or modify the project

Please visit the official GitHub repository:

https://github.com/azurite-bedrock/Sweep-N-Slash

All development happens there. Pull requests and contributions are welcome
through GitHub as well.

Thank you!
